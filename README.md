# noto

日記・知識DB・カレンダー・ダッシュボードが統合された個人向けWebアプリです。

## 機能

- **日記を書く** — タイトル・本文・カテゴリ・気分・タグで記録
- **知識データベース** — 日記から気づきを即座に登録・管理
- **日記一覧** — 記録した日記をリスト表示・詳細確認
- **検索・フィルター** — キーワード検索、カテゴリ・タグ・気分で絞り込み
- **カレンダー表示** — 月表示で記録日を視覚化、日付クリックで詳細表示
- **ダッシュボード** — 月別投稿数グラフ、カテゴリ分布、タグクラウド、気分の統計
- **ローカル保存** — localStorage でデータを自動永続化

## インストール

```bash
# リポジトリをクローン
git clone https://github.com/yourusername/noto.git
cd noto

# 依存パッケージをインストール
npm install

# 開発サーバーを起動
npm start
```

ブラウザで http://localhost:3000 が自動的に開きます。

## 技術スタック

- **React 18** — UI構築
- **Chart.js** — グラフ表示（棒グラフ、ドーナツグラフ）
- **CSS3** — スタイリング（レスポンシブ対応）
- **localStorage** — クライアント側データ永続化

## 使い方

### 日記を書く
1. 左のナビゲーションから「日記を書く」を選択
2. タイトル・内容・カテゴリ・気分を入力
3. タグを追加（Enterキーで確定）
4. 右パネルの「知識を追加する」で気づきを同時記録
5. 「保存する」をクリック

### 日記を検索
1. 「検索・フィルター」を選択
2. キーワード入力、またはカテゴリ・タグ・気分で絞り込み
3. 結果をクリックして詳細表示

### カレンダーで確認
1. 「カレンダー」を選択
2. 月・週を変更して記録状況を視覚化
3. 日付をクリックして下部に詳細を表示

### ダッシュボード
1. 「ダッシュボード」を選択
2. 統計情報とグラフで記録の傾向を確認
3. タグクラウドから使用頻度を把握

## データ保存

すべてのデータはブラウザの localStorage に保存されます。
- キー: `noto_entries` (日記), `noto_kb` (知識DB)
- ブラウザのキャッシュを削除するとデータが失われます

## カスタマイズ

### カテゴリの変更
`src/data/constants.js` の `CATEGORIES` 配列を編集：

```javascript
export const CATEGORIES = ['仕事', '学習', '日常', '健康'];
```

### カラーテーマの変更
同ファイルの `CAT_STYLES` で各カテゴリのカラーを変更。
